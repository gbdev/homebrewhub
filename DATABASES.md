# Databases

Homebrew Hub aggregates entries from multiple community-maintained databases, each covering a different platform. Each database is a standalone repository storing game metadata as JSON files along with binaries, and is periodically synced into Homebrew Hub.

Entry metadata follows the Homebrew Hub [Entry JSON schema](https://github.com/gbdev/database/tree/master/schemas).

| Repository | Platforms | License | Entries (approx.) |
|---|---|---|---|
| [gbdev/database](https://github.com/gbdev/database) | Game Boy, Game Boy Color | GPL-3.0 | ~1300 |
| [gbadev-org/games](https://github.com/gbadev-org/games) | Game Boy Advance | GPL-3.0 | 200 |
| [nesdev-org/homebrew-db](https://github.com/nesdev-org/homebrew-db) | NES | GPL-3.0 | ~23 |
